# IT Inventory System — Demo Template Page

## Overview

Create a full demo template page for an IT Asset Management System with a dynamic sidebar, paginated datatable, modals, export functionality, and scroll animations. Built on Next.js 16 + Tailwind CSS v4 + shadcn/ui + TanStack Table.

---

## Phase 1: Setup & Installation

### 1.1 Initialize shadcn/ui

```bash
npx shadcn@latest init
```

- Style: Default
- Base color: Neutral
- CSS variables: Yes

### 1.2 Install shadcn components

```bash
npx shadcn@latest add button dialog input table select dropdown-menu sheet sidebar card badge separator label textarea tabs popover scroll-area checkbox tooltip collapsible alert-dialog sonner breadcrumb
```

### 1.3 Install npm packages

```bash
npm install @tanstack/react-table framer-motion jspdf jspdf-autotable xlsx date-fns lucide-react clsx tailwind-merge class-variance-authority
```

## Phase 5: DataTable Features

### Columns (IT Asset fields)
| Column | Sortable | Filterable | Notes |
|--------|----------|------------|-------|
| Asset Tag | Yes | Yes | `AST-0001` format |
| Name | Yes | No | Device description |
| Type | Yes | Faceted (dropdown) | Laptop, Desktop, Monitor, etc. |
| Brand / Model | Yes | No | Combined display |
| Status | Yes | Faceted (dropdown) | Badge with color coding |
| Assigned To | Yes | No | "Unassigned" fallback |
| Location | Yes | Faceted (dropdown) | Office locations |
| Purchase Date | Yes | No | date-fns formatted |
| Warranty Expiry | Yes | No | Highlighted if expired |
| Actions | N/A | N/A | Dropdown: View, Edit, Delete, Restore |

### Search & Filter Behavior
- **Search box**: Simulated backend query (debounced, 300ms) → sets TanStack `globalFilter`
- **Advanced Search button**: Opens dialog with multiple field inputs → simulated backend query
- **Faceted filters** (Status, Type, Location dropdowns): Client-side filtering on loaded data via TanStack `columnFilters`
- **Advanced Filter button**: Opens dialog with dynamic condition builder (field + operator + value, AND logic) → client-side filtering on loaded data

### Export
- **PDF**: `jspdf` + `jspdf-autotable` — renders visible columns with title and date
- **Excel**: `xlsx` (SheetJS) — creates workbook with auto-width columns

### Pagination
- Page sizes: 10, 25, 50, 100
- First/Prev/Next/Last buttons
- "Showing X to Y of Z rows"
