# Add Location Field to Assignment Module

## Overview
Add a `location_id` field to the assignment module. When an assignment is created or updated with a location, the asset's `location_id` is also updated.

## Files to Modify

### 1. `lib/types/assignment.ts`
- Add `location_id?: string | null` to `Assignment` interface
- Add `location_name?: string` to `Assignment` interface
- Add `location_id?: string` to `CreateAssignmentInput` interface

### 2. `lib/db/models/assignment.ts`
- Add `location_id` field (ObjectId ref "Location", default null) to `IAssignment` interface and schema
- Add index on `location_id`

### 3. `lib/services/assignment-service.ts`
- `toAssignment`: Add `location_id` mapping
- `enrichAssignment`: Fetch Location and add `location_name`
- `getAssignmentSelectOptions`: Import Location model, query active locations, return `locations: { id: string; name: string }[]`
- `createAssignment`: Include `location_id` in created assignment; update asset's `location_id`
- `updateAssignment`: Include `location_id` in update; update asset's `location_id`

### 4. `lib/actions/assignment-actions.ts`
- Update `getAssignmentSelectOptions` return type to include `locations`

### 5. `components/modals/assignment-form-modal.tsx`
- Add `locations` state and fetch in `getAssignmentSelectOptions`
- Add `location_id` to `defaultFormData`
- Populate `location_id` from `assignment` in useEffect
- Add Location `<Select>` dropdown in the form (grid row with Assigned Date)
- Add `w-full` to all SelectTriggers per project rules

### 6. `components/modals/assignment-view-modal.tsx`
- Add Location display in the left column (below Department)

### 7. `components/data-table/assignment-data-table-columns.tsx`
- Add Location column after Department column

### 8. `app/api/assignments/route.ts` (POST)
- Pass `location_id` from body to `createAssignment`

### 9. `app/api/assignments/[id]/route.ts` (PUT)
- Pass `location_id` from body to `updateAssignment`

## Implementation Order
1. Types (`lib/types/assignment.ts`)
2. Model (`lib/db/models/assignment.ts`)
3. Service (`lib/services/assignment-service.ts`)
4. Actions (`lib/actions/assignment-actions.ts`)
5. API routes (`app/api/assignments/...`)
6. Form modal (`components/modals/assignment-form-modal.tsx`)
7. View modal (`components/modals/assignment-view-modal.tsx`)
8. Data table columns (`components/data-table/assignment-data-table-columns.tsx`)
