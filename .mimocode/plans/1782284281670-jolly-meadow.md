# Ticket Detail Page Improvements

## Changes

### 1. Breadcrumb: Replace [id] with ticket number

**Files to modify:**
- `components/layout/header.tsx` — consume `useBreadcrumbOverrides()` and merge overrides into breadcrumb labels
- `app/(dashboard)/tickets/[id]/page.tsx` — call `setBreadcrumbOverrides()` on load to replace `[id]` with `ticket.ticket_no`

**Approach:**
- Create a `BreadcrumbOverrideProvider` context (`components/layout/breadcrumb-override-context.tsx`) that provides `setBreadcrumbOverrides(overrides: Record<string, string>)` where keys are the URL segment value to replace and values are the display label.
- Wrap the app layout (`app/(dashboard)/layout.tsx`) with the provider.
- In `header.tsx`, consume the context and apply overrides when building `breadcrumbItems` — if a segment matches an override key, use the override value as the label.
- In the ticket detail page, once the ticket loads, call `setBreadcrumbOverrides({ [ticket.id]: ticket.ticket_no })`.
