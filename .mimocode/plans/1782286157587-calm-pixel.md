# Plan: Add "Assign" Button to Ticket Detail Page

## Summary
Add an "Assign" button next to "Update Status" (renamed to "Status") in the ticket detail sidebar. Clicking it opens a modal to select/transfer a technician or administrator.

## Key Findings
- `updateTicket()` already accepts `assigned_to` in `UpdateTicketInput` — no service changes needed
- `getTicketSelectOptions()` already returns `users` (filtered by Administrator/Technician roles) — but the page's `selectOptions` type doesn't include it yet
- `updateTicket` service already handles: assignee-change email notifications, status logs for assignment changes
- Existing modals follow a consistent Dialog pattern

## Changes

- Select dropdown with "Unassigned" option + all users (sorted A-Z)
- Uses the scrollable modal pattern from AGENTS.md
- Icon: `UserPlus` from lucide-react

- Add "Assign" button next to "Status" button (same row, same styling)
