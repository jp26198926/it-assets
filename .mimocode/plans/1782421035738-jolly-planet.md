# Fix: MissingSchemaError for "Item" model in dashboard-service

## Root Cause
`lib/services/dashboard-service.ts` calls `.populate("item_id")` (line 63), which requires the `Item` model to be registered in Mongoose. The service dynamically imports `category` and `uom` models (lines 34-35) but never imports the `Item` model.

## Fix
**File**: `lib/services/dashboard-service.ts`

Add one dynamic import after line 35:
```ts
await import("@/lib/db/models/item");
```

This follows the existing pattern used for `Category` and `UOM`.
