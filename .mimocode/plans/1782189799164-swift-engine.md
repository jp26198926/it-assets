# Fix: Image Loading State

## Problem
`onLoadingComplete` doesn't fire reliably for `unoptimized` next/image, so the image stays hidden.

## Solution
Remove React state. Use a pure CSS approach: skeleton behind image, image fades in with CSS transition.

## File
- `components/modals/item-view-modal.tsx`

## Changes
1. Remove `useState(true)` and `imageLoading` state
2. Remove `import { useState }` (if only used for this)
3. Replace image section with CSS-only approach:
   - Wrapper div with `relative` 
   - Skeleton div with `absolute animate-pulse`
   - Image with `opacity-0` initially, using CSS `onLoad` to add `opacity-100`

## Verification
- Skeleton shows briefly, then image fades in
- No React state issues on modal reopen
